About Scientific Calcii:
  • Please Input Valid Arithmatic Operations for Calculating Values.
  • AC is used for All Clear and DEL is used for deleting one value.
  • For Trigonometric Values of sin, cos and tan; first input the angle value in radian as {Degree Value*(pi/180)} than click on desired trigonometic function.
  • For other Scientific values, first input value than click on the function like log, root, power exponential and factorial.
  • Root function only gives square-root value for input value.
    Power function only gives the square value for input value.
